### Reel Best Practices

**Hook (First 3 seconds):**
- Start with the problem or a surprising fact
- Use text overlay for silent viewers
- Eye-catching visual immediately

**Middle (3-15 seconds):**
- Show the solution
- Quick, clear demonstration
- Keep it moving (no static shots)

**End (Last 3 seconds):**
- Clear CTA
- Link in bio reminder
- Brand name visible

**Technical:**
- Vertical format (9:16 ratio)
- Text overlays for silent viewing
- Trending audio (but make sure it fits)
- Captions with keywords
- Location tag (if relevant)
